## Overview

This project was originally developed for a client to streamline payment tracking across multiple resorts. To make it available for public use, **all confidential data have been removed**, and sample data has been included in the dashboard.

Managing resort payments manually can be time-consuming and error-prone, especially when handling multiple resorts, payment types, and daily updates. This Excel dashboard provides a **dynamic, consolidated solution** that saves time and improves visibility.

## Core Features

- **Collection by Resort:** Quickly view total payments and outstanding balances per resort.
- **Payment Status Pie Chart:** Visualize the proportion of payments (fully paid, deposits, pending).
- **Deposit vs Full Payment Pie Chart:** Understand the share of deposits versus full payments.
- **Top 10 Countries Bar Chart:** Highlight the top countries contributing to revenue.
- **Dynamic & Auto-Update:** Add new data to the tables, and the dashboard updates automatically.
- **Interactive Filters:** Filter by month or resort for detailed insights.
